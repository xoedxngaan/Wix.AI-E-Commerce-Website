# Wix-E-Commerce-Website
Created an AI-assisted e-commerce website using Wix. The project explores web application development, system workflows, and basic data handling processes while demonstrating how AI tools can support rapid digital product development. 
See link to my site: https://zoedingaan.wixsite.com/princess-cyberverse 

Project: Princess Cyberverse - AI-Assisted E-commerce Brand Launch
Goal: To design, build, and launch a digital product storefront targeting a niche audience interested in aesthetic digital planning tools.

My Role & Process:
· Concept & Branding: Developed the "Princess Cyberverse" brand identity, including visual aesthetic (Y2K/soft-girl), tone of voice, and target audience definition.

· Platform Development: Built a fully functional e-commerce site using Wix, integrating product listings, a digital download fulfillment system, and an email marketing sign-up workflow.

· AI Integration: Leveraged AI tools to accelerate copywriting for product descriptions, brand storytelling, and FAQ content, allowing for rapid iteration and refinement.

· UX & Workflow Design: Mapped the end-to-end customer journey, from landing on the homepage to post-purchase support, resulting in a clear FAQ section and streamlined user experience.

Key Outcomes: Launched a live, branded e-commerce site with featured products, customer testimonials, and clear systems for customer engagement and digital delivery. The project demonstrates the ability to blend creative direction with technical execution and AI-assisted workflows.
